# Nombre del Proyecto

Este proyecto tiene como objetivo obtener y analizar datos de fútbol de la Liga Española a través de una API, enfocándose en las estadísticas individuales de un jugador específico dentro de un equipo para la temporada solicitada. Se recogen datos como minutos jugados, goles, asistencias y tarjetas.

Además, se genera una gráfica que muestra la evolución del valor de mercado del jugador seleccionado durante los últimos 5 años. 
El proyecto permite explorar tanto el rendimiento deportivo como la variación en el valor de mercado del jugador a lo largo del tiempo, proporcionando una visión completa de su trayectoria y relevancia en el fútbol.
